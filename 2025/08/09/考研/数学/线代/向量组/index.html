向量组

概念要点

向量组基础

  • 向量定义
    • n维向量：n个数的有序数组，行向量α=[a1,a2,⋯ ,an]\alpha = [a_1, a_2, \cdots, a_n]α=[a1​,a2​,⋯,an​]，列向量αT=[a1,a2,⋯ ,an]T\alpha^T = [a_1, a_2, \cdots, a_n]^TαT=[a1​,a2​,⋯,an​]T
    • 可视为1×n1 \times n1×n或n×1n \times 1n×1矩阵
  • 向量运算
    • 相等：α=β⇔ai=bi (i=1,2,⋯ ,n)\alpha = \beta \Leftrightarrow a_i = b_i \ (i=1,2,\cdots,n)α=β⇔ai​=bi​ (i=1,2,⋯,n)
    • 加法：α+β=[a1+b1,a2+b2,⋯ ,an+bn]\alpha + \beta = [a_1 + b_1, a_2 + b_2, \cdots, a_n + b_n]α+β=[a1​+b1​,a2​+b2​,⋯,an​+bn​]
    • 数乘：kα=[ka1,ka2,⋯ ,kan]k\alpha = [ka_1, ka_2, \cdots, ka_n]kα=[ka1​,ka2​,⋯,kan​]（kkk为实数）
  • 内积与正交
    • 内积：(α,β)=αTβ=∑i=1naibi=a1b1+a2b2+⋯+anbn(\alpha, \beta) = \alpha^T \beta = \sum_{i=1}^n a_i b_i = a_1b_1 + a_2b_2 + \cdots + a_nb_n(α,β)=αTβ=∑i=1n​ai​bi​=a1​b1​+a2​b2​+⋯+an​bn​
    • 正交：αTβ=0\alpha^T \beta = 0αTβ=0时，α\alphaα与β\betaβ正交
    • 模长：∥α∥=a12+a22+⋯+an2=(α,α)\|\alpha\| = \sqrt{a_1^2 + a_2^2 + \cdots + a_n^2} = \sqrt{(\alpha, \alpha)}∥α∥=a12​+a22​+⋯+an2​​=(α,α)​
    • 单位向量：∥α∥=1\|\alpha\| = 1∥α∥=1时，α\alphaα为单位向量
    • 标准正交组：αiTαj={1,i=j0,i≠j\alpha_i^T \alpha_j = \begin{cases} 1, & i=j \\ 0, & i \neq j \end{cases}αiT​αj​={1,0,​i=ji=j​
  • 正交矩阵
    • 定义：n阶方阵AAA满足ATA=EA^T A = EATA=E
    • 充要条件：AT=A−1A^T = A^{-1}AT=A−1；行（列）向量组是规范正交基
    • 性质：作用于向量时模长不变（如旋转矩阵[cos⁡θ−sin⁡θsin⁡θcos⁡θ]\begin{bmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{bmatrix}[cosθsinθ​−sinθcosθ​]）

线性表示与相关性

  • 核心概念
    • 线性组合：k1α1+k2α2+⋯+kmαmk_1\alpha_1 + k_2\alpha_2 + \cdots + k_m\alpha_mk1​α1​+k2​α2​+⋯+km​αm​（kik_iki​为常数）
    • 线性表示：β=k1α1+⋯+kmαm\beta = k_1\alpha_1 + \cdots + k_m\alpha_mβ=k1​α1​+⋯+km​αm​（β\betaβ在α\alphaα张成空间中）
    • 线性相关：存在不全为零的kik_iki​，使k1α1+⋯+kmαm=0k_1\alpha_1 + \cdots + k_m\alpha_m = 0k1​α1​+⋯+km​αm​=0（至少一向量可由其余表示）
    • 线性无关：仅当kik_iki​全为零时，k1α1+⋯+kmαm=0k_1\alpha_1 + \cdots + k_m\alpha_m = 0k1​α1​+⋯+km​αm​=0（无多余向量）
  • 判别七大定理
    • 定理1：向量组（n≥2n \geq 2n≥2）相关⇔\Leftrightarrow⇔至少一向量可由其余表示
    • 定理2：α1,⋯ ,αn\alpha_1,\cdots,\alpha_nα1​,⋯,αn​无关，β,α1,⋯ ,αn\beta,\alpha_1,\cdots,\alpha_nβ,α1​,⋯,αn​相关⇒\Rightarrow⇒ β\betaβ可由唯一表示
    • 定理3：β1,⋯ ,βt\beta_1,\cdots,\beta_tβ1​,⋯,βt​可由α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​表示，且t>st > st>s ⇒\Rightarrow⇒ β1,⋯ ,βt\beta_1,\cdots,\beta_tβ1​,⋯,βt​相关
    • 定理4：相关⇔\Leftrightarrow⇔齐次方程组有非零解（r([α1,⋯ ,αm])<mr([\alpha_1,\cdots,\alpha_m]) < mr([α1​,⋯,αm​])<m）；无关⇔\Leftrightarrow⇔仅零解（r=mr = mr=m）
    • 定理5：β\betaβ可由α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​表示⇔\Leftrightarrow⇔ r([α1,⋯ ,αs])=r([α1,⋯ ,αs,β])r([\alpha_1,\cdots,\alpha_s]) = r([\alpha_1,\cdots,\alpha_s, \beta])r([α1​,⋯,αs​])=r([α1​,⋯,αs​,β])
    • 定理6：部分相关⇒\Rightarrow⇒整体相关；整体无关⇒\Rightarrow⇒部分无关
    • 定理7：无关向量延长后仍无关；相关向量缩短后仍相关

极大无关组与秩

  • 极大线性无关组
    • 定义：①线性无关；②组内任一向量可由其表示（不唯一）
    • 求法：列向量构矩阵→\rightarrow→初等行变换→\rightarrow→行阶梯形中非零行首列对应向量
  • 向量组的秩
    • 定义：极大无关组含向量个数（r(α1,⋯ ,αs)=rr(\alpha_1,\cdots,\alpha_s) = rr(α1​,⋯,αs​)=r）
    • 三秩相等：$r(A) = 行秩（行向量组的秩）行秩（行向量组的秩）行秩（行向量组的秩） = $列秩（列向量组的秩）
  • 等价向量组
    • 定义：可相互线性表示（反身性、对称性、传递性）
    • 判别：r(I)=r(II)=r(I,II)r(I) = r(II) = r(I, II)r(I)=r(II)=r(I,II)
    • 与等价矩阵区别：矩阵等价需同型且r(A)=r(B)r(A) = r(B)r(A)=r(B)；向量组等价需同维且相互表示

向量空间（仅数学一）

  • 基本概念
    • 基：RnR^nRn中线性无关的有序向量组ξ1,⋯ ,ξn\xi_1,\cdots,\xi_nξ1​,⋯,ξn​（张成空间）
    • 维数：基中向量个数（空间维度为nnn）
    • 坐标：α=a1ξ1+⋯+anξn\alpha = a_1\xi_1 + \cdots + a_n\xi_nα=a1​ξ1​+⋯+an​ξn​，坐标为[a1,⋯ ,an]T[a_1,\cdots,a_n]^T[a1​,⋯,an​]T
  • 基变换与坐标变换
    • 基变换公式：[η1,⋯ ,ηn]=[ξ1,⋯ ,ξn]C[\eta_1, \cdots, \eta_n] = [\xi_1, \cdots, \xi_n]C[η1​,⋯,ηn​]=[ξ1​,⋯,ξn​]C（CCC为过渡矩阵，可逆）
    • 坐标变换公式：x=Cyx = Cyx=Cy或y=C−1xy = C^{-1}xy=C−1x（x,yx,yx,y为同一向量在不同基下的坐标）
  • 施密特正交化
    • 正交化：β1=α1\beta_1 = \alpha_1β1​=α1​；β2=α2−(α2,β1)(β1,β1)β1\beta_2 = \alpha_2 - \frac{(\alpha_2, \beta_1)}{(\beta_1, \beta_1)}\beta_1β2​=α2​−(β1​,β1​)(α2​,β1​)​β1​
    • 单位化：γi=βi∥βi∥\gamma_i = \frac{\beta_i}{\|\beta_i\|}γi​=∥βi​∥βi​​（规范正交基）

解题策略

一、向量组线性相关性的判别

题型特征：

判断给定向量组α1,α2,⋯ ,αm\alpha_1,\alpha_2,\cdots,\alpha_mα1​,α2​,⋯,αm​是否线性相关（或无关）。

解题策略与技巧：

  1. 定义法：
    设k1α1+k2α2+⋯+kmαm=0k_1\alpha_1 + k_2\alpha_2 + \cdots + k_m\alpha_m = 0k1​α1​+k2​α2​+⋯+km​αm​=0，转化为齐次线性方程组，若存在不全为零的kik_iki​使方程成立，则线性相关；否则无关。

  2. 行列式法（仅适用于nnn个nnn维向量）：
    构造矩阵A=[α1,α2,⋯ ,αn]A = [\alpha_1, \alpha_2, \cdots, \alpha_n]A=[α1​,α2​,⋯,αn​]，计算行列式∣A∣|A|∣A∣：

    • 若∣A∣=0|A| = 0∣A∣=0，则向量组线性相关；
    • 若∣A∣≠0|A| \neq 0∣A∣=0，则线性无关。

  3. 矩阵秩法：
    构造矩阵A=[α1,α2,⋯ ,αm]A = [\alpha_1, \alpha_2, \cdots, \alpha_m]A=[α1​,α2​,⋯,αm​]，求秩r(A)r(A)r(A)：

    • 若r(A)<mr(A) < mr(A)<m，则线性相关；
    • 若r(A)=mr(A) = mr(A)=m，则线性无关。

  4. 定理法：

    • 含零向量或成比例向量的组必相关；
    • 部分相关⇒\Rightarrow⇒整体相关；整体无关⇒\Rightarrow⇒部分无关；
    • 若向量组β1,⋯ ,βt\beta_1,\cdots,\beta_tβ1​,⋯,βt​可由α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​表示且t>st > st>s，则β1,⋯ ,βt\beta_1,\cdots,\beta_tβ1​,⋯,βt​相关（以少表多，多的相关）；
    • 向量个数超过维数（m>nm > nm>n）时必相关。

易错点：

  • 混淆“线性相关”与“线性表示”的关系：线性相关仅需存在非零系数，未必是某一向量可由其余表示（但n≥2n \geq 2n≥2时等价，见定理1）；
  • 忽略“向量个数与维数”的关系：n+1n+1n+1个nnn维向量必相关，不可误用行列式法（行列式不存在）。

二、向量的线性表示判定及表达式求解

题型特征：

判断向量β\betaβ能否由向量组α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​线性表示，若能，求出表达式β=k1α1+⋯+ksαs\beta = k_1\alpha_1 + \cdots + k_s\alpha_sβ=k1​α1​+⋯+ks​αs​。

解题策略与技巧：

  1. 秩条件判定：
    构造矩阵A=[α1,⋯ ,αs]A = [\alpha_1, \cdots, \alpha_s]A=[α1​,⋯,αs​]（系数矩阵）和A‾=[A,β]\overline{A} = [A, \beta]A=[A,β]（增广矩阵）：

    • 若r(A)=r(A‾)r(A) = r(\overline{A})r(A)=r(A)，则可表示；
    • 若r(A)≠r(A‾)r(A) \neq r(\overline{A})r(A)=r(A)，则不可表示。

  2. 表达式求解：
    对A‾\overline{A}A作初等行变换化为行阶梯形，确定自由变量（若存在），回代得系数kik_iki​。

  3. 特殊情形：

    • 若α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​线性无关，且β,α1,⋯ ,αs\beta,\alpha_1,\cdots,\alpha_sβ,α1​,⋯,αs​线性相关，则β\betaβ可由唯一表示（定理2）。

易错点：

  • 未验证秩条件直接判定“可表示”；
  • 求解表达式时未将矩阵化为行最简形，导致系数计算错误。

三、极大线性无关组的求解与向量组秩的计算

题型特征：

求向量组α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​的极大线性无关组，并计算秩r(α1,⋯ ,αs)r(\alpha_1,\cdots,\alpha_s)r(α1​,⋯,αs​)，将其余向量用极大组表示。

解题策略与技巧：

  1. 步骤：

    • 列向量构造矩阵A=[α1,α2,⋯ ,αs]A = [\alpha_1, \alpha_2, \cdots, \alpha_s]A=[α1​,α2​,⋯,αs​]；
    • 对AAA作初等行变换化为行阶梯形，确定非零行数rrr（即秩）；
    • 选取每行首非零元对应的原列向量，即为极大线性无关组；
    • 进一步化为行最简形，回代得其余向量的表示式。

  2. 技巧：

    • 极大无关组不唯一，但所含向量个数唯一（等于秩）；
    • 行变换不改变列向量的线性相关性，故可通过行最简形直接读系数。

易错点：

  • 混用初等行变换与列变换：求极大无关组时只能用行变换（或只能用列变换），不可混用；
  • 误将“行向量组”按列向量处理：行向量组应构造矩阵后作初等列变换，原理相同但操作方向需一致。

四、等价向量组的判定

题型特征：

判断两向量组（I）α1,⋯ ,αs\alpha_1,\cdots,\alpha_sα1​,⋯,αs​与（II）β1,⋯ ,βt\beta_1,\cdots,\beta_tβ1​,⋯,βt​是否等价（相互线性表示）。

解题策略与技巧：

  1. 等价条件：

    • 向量组（I）与（II）可相互表示；
    • 秩相等且其中一组可由另一组表示，即r(I)=r(II)=r(I,II)r(I) = r(II) = r(I, II)r(I)=r(II)=r(I,II)（三秩相等）。

  2. 判定步骤：

    • 构造矩阵[A,B]=[α1,⋯ ,αs,β1,⋯ ,βt][A, B] = [\alpha_1,\cdots,\alpha_s, \beta_1,\cdots,\beta_t][A,B]=[α1​,⋯,αs​,β1​,⋯,βt​]；
    • 求r(A)r(A)r(A)、r(B)r(B)r(B)、r([A,B])r([A, B])r([A,B])，若三者相等则等价。

易错点：

  • 混淆“等价向量组”与“等价矩阵”：等价矩阵需同型且秩相等，等价向量组需同维且相互表示，二者无必然联系；
  • 仅验证“秩相等”即判定等价，忽略“相互表示”的条件。

五、正交矩阵与正交向量组相关问题

题型特征：

判断矩阵是否为正交矩阵，或构造正交向量组（施密特正交化）。

解题策略与技巧：

  1. 正交矩阵判定：
    对nnn阶矩阵AAA，验证ATA=EA^T A = EATA=E（或AAT=EA A^T = EAAT=E，或A−1=ATA^{-1} = A^TA−1=AT），或其行（列）向量组为规范正交基（αiTαj=δij\alpha_i^T \alpha_j = \delta_{ij}αiT​αj​=δij​，δij=1\delta_{ij}=1δij​=1若i=ji=ji=j，否则0）。

  2. 施密特正交化：
    对线性无关向量组α1,α2\alpha_1,\alpha_2α1​,α2​，正交化公式：

    β1=α1,β2=α2−(α2,β1)(β1,β1)β1\beta_1 = \alpha_1, \quad \beta_2 = \alpha_2 - \frac{(\alpha_2, \beta_1)}{(\beta_1, \beta_1)} \beta_1 β1​=α1​,β2​=α2​−(β1​,β1​)(α2​,β1​)​β1​

    单位化：γi=βi∥βi∥\gamma_i = \frac{\beta_i}{\|\beta_i\|}γi​=∥βi​∥βi​​（∥βi∥=(βi,βi)\|\beta_i\| = \sqrt{(\beta_i, \beta_i)}∥βi​∥=(βi​,βi​)​）。

易错点：

  • 正交矩阵判定时仅验证行（或列）向量正交，忽略“单位向量”条件；
  • 施密特正交化计算错误，尤其内积和模长的计算。

六、向量空间（仅数学一）：基变换与坐标变换

题型特征：

求两个基之间的过渡矩阵，或同一向量在不同基下的坐标。

解题策略与技巧：

  1. 基变换公式：
    设基（I）ξ1,⋯ ,ξn\xi_1,\cdots,\xi_nξ1​,⋯,ξn​，基（II）η1,⋯ ,ηn\eta_1,\cdots,\eta_nη1​,⋯,ηn​，则[η1,⋯ ,ηn]=[ξ1,⋯ ,ξn]C[\eta_1,\cdots,\eta_n] = [\xi_1,\cdots,\xi_n] C[η1​,⋯,ηn​]=[ξ1​,⋯,ξn​]C，其中CCC为过渡矩阵，且C=[ξ1,⋯ ,ξn]−1[η1,⋯ ,ηn]C = [\xi_1,\cdots,\xi_n]^{-1} [\eta_1,\cdots,\eta_n]C=[ξ1​,⋯,ξn​]−1[η1​,⋯,ηn​]。

  2. 坐标变换公式：
    若向量α\alphaα在基（I）、（II）下的坐标为x,yx, yx,y，则x=Cyx = C yx=Cy或y=C−1xy = C^{-1} xy=C−1x。

易错点：

  • 过渡矩阵的方向混淆：是从基（I）到基（II），而非相反；
  • 求逆矩阵错误导致过渡矩阵计算错误。

七、秩的不等式相关证明（如r(AB)≤min⁡{r(A),r(B)}r(AB) \leq \min\{r(A), r(B)\}r(AB)≤min{r(A),r(B)}）

题型特征：

证明与矩阵或向量组秩相关的不等式（如r(A+B)≤r([A,B])≤r(A)+r(B)r(A+B) \leq r([A,B]) \leq r(A) + r(B)r(A+B)≤r([A,B])≤r(A)+r(B)）。

解题策略与技巧：

  • 利用“向量组秩的性质”：若βi\beta_iβi​可由αj\alpha_jαj​表示，则r(β1,⋯ ,βt)≤r(α1,⋯ ,αs)r(\beta_1,\cdots,\beta_t) \leq r(\alpha_1,\cdots,\alpha_s)r(β1​,⋯,βt​)≤r(α1​,⋯,αs​)；
  • 构造分块矩阵，通过初等变换分析秩的关系。

易错点：

  • 忽略“向量组表示关系”与“秩大小”的关联，直接套用公式而不证明。
