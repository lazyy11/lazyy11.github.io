第十六讲 无穷级数

概念

考题与题型

  • 考题：级数敛散性判别、幂级数及收敛域、幂级数求和函数、函数展开成幂级数；傅里叶级数（仅数学一）
  • 题型：选择题、填空题、解答题
  • 重难点：判别级数敛散性、求幂级数和函数、函数展开成幂级数；泰勒级数收敛条件（仅数学一）；傅里叶级数展开与和函数（仅数学一）

基础知识结构

  • 引例：无穷多项相加与有限项的区别（芝诺悖论）
  • 常数项级数：概念与性质、敛散性判别
  • 幂级数：概念、收敛域、求和函数
  • 函数展开成幂级数：直接法、间接法
  • 傅里叶级数（仅数学一）：周期级数、正弦/余弦级数、狄利克雷定理

常数项级数

  • 概念与性质
    • 概念：无穷级数定义（∑n=1∞un\sum_{n=1}^{\infty}u_n∑n=1∞​un​）、部分和数列（Sn=∑k=1nukS_n=\sum_{k=1}^nu_kSn​=∑k=1n​uk​）
    • 敛散性：lim⁡n→∞Sn\lim_{n\to\infty}S_nlimn→∞​Sn​存在则收敛，否则发散
    • 性质：线性组合收敛性、改变有限项不改变敛散性、加括号收敛性、通项极限为0（必要条件）
  • 敛散性判别
    • 正项级数：收敛原则（部分和有界）；判别法（比较、极限形式、比值、根值、积分）
    • 交错级数：莱布尼茨判别法（unu_nun​单调不增且lim⁡un=0\lim u_n=0limun​=0）；绝对收敛判别
    • 任意项级数：绝对收敛（∑∣un∣\sum|u_n|∑∣un​∣收敛）、条件收敛（∑un\sum u_n∑un​收敛但∑∣un∣\sum|u_n|∑∣un​∣发散）

幂级数

  • 概念：∑n=0∞anxn\sum_{n=0}^{\infty}a_nx^n∑n=0∞​an​xn或∑n=0∞an(x−x0)n\sum_{n=0}^{\infty}a_n(x-x_0)^n∑n=0∞​an​(x−x0​)n
  • 收敛域
    • 阿贝尔定理：收敛点内绝对收敛，发散点外发散
    • 收敛半径：R=1ρR=\frac{1}{\rho}R=ρ1​（ρ=lim⁡∣an+1an∣\rho=\lim|\frac{a_{n+1}}{a_n}|ρ=lim∣an​an+1​​∣或lim⁡∣an∣n\lim\sqrt[n]{|a_n|}limn∣an​∣​）
    • 收敛区间与域：(−R,R)(-R,R)(−R,R)，端点单独判别（绝对收敛，条件收敛，发散都有可能）
  • 求和函数
    • 小猪配齐法：x的阶数往n靠
    • 运算法则：线性运算、乘积（柯西乘积）
    • 恒等变形：通项与下标变换
    • 性质：连续、逐项积分（收敛域可能扩大）、逐项求导（收敛域可能缩小）
    • 重要展开式：exe^xex、11±x\frac{1}{1\pm x}1±x1​、ln⁡(1+x)\ln(1+x)ln(1+x)、sin⁡x\sin xsinx、cos⁡x\cos xcosx、(1+x)α(1+x)^\alpha(1+x)α

解题策略

级数专题解题策略总结（分板块突破）

一、常数项级数敛散性判别

核心逻辑：先判类型（正项/交错/任意项），再选方法，从“通项极限→判别法”分步推进。

1. 第一步：通项极限预判

若 lim⁡n→∞un≠0\lim_{n\to\infty} u_n \neq 0limn→∞​un​=0，直接判定 级数发散（必要条件不满足）；
若 lim⁡n→∞un=0\lim_{n\to\infty} u_n = 0limn→∞​un​=0，进入下一步。

2. 分类判别

  • 正项级数（un≥0u_n \geq 0un​≥0）：

    • 比较判别法：
      • 非极限形式：找已知级数（ppp级数、几何级数）作“上界/下界”，或用等价无穷小替换（n→∞n\to\inftyn→∞ 时，un∼vnu_n \sim v_nun​∼vn​，则 ∑un\sum u_n∑un​ 与 ∑vn\sum v_n∑vn​ 同敛散）。
      • 极限形式
    • 比值法（lim⁡n→∞un+1un=ρ\boldsymbol{\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = \rho}limn→∞​un​un+1​​=ρ）：ρ<1\rho < 1ρ<1 收敛，ρ>1\rho > 1ρ>1 发散，ρ=1\rho = 1ρ=1 失效（换比较法）。
    • 根值法（lim⁡n→∞unn=ρ\boldsymbol{\lim_{n\to\infty} \sqrt[n]{u_n} = \rho}limn→∞​nun​​=ρ）：适用幂次型（如 un=(n/(2n+1))nu_n = (n/(2n+1))^nun​=(n/(2n+1))n），结论同比值法。
    • 积分法：若 un=f(n)u_n = f(n)un​=f(n) 且 f(x)f(x)f(x) 非负递减可积，则 ∑un\sum u_n∑un​ 与 ∫1+∞f(x)dx\int_1^{+\infty} f(x)dx∫1+∞​f(x)dx 同敛散。

  • 交错级数（un=(−1)nvnu_n = (-1)^n v_nun​=(−1)nvn​，vn>0v_n > 0vn​>0）：
    用 莱布尼茨判别法：需满足
    ① vn\boldsymbol{①\ v_n}① vn​ 单调递减（vn+1≤vnv_{n+1} \leq v_nvn+1​≤vn​，可求导 f(x)=vxf(x) = v_xf(x)=vx​ 判单调性）；
    ② lim⁡n→∞vn=0\boldsymbol{②\ \lim_{n\to\infty} v_n = 0}② limn→∞​vn​=0。
    若满足，级数收敛（可能条件收敛）。

  • 任意项级数（含正负项，非交错）：
    先判 绝对收敛（即 ∑∣un∣\sum |u_n|∑∣un​∣ 的敛散，按正项级数方法）：

    • 若 ∑∣un∣\sum |u_n|∑∣un​∣ 收敛，则 ∑un\sum u_n∑un​ 绝对收敛；
    • 若 ∑∣un∣\sum |u_n|∑∣un​∣ 发散，再判 ∑un\sum u_n∑un​ 本身是否收敛（如交错部分用莱布尼茨，或拆分为正项和负项分析）。

二、幂级数与收敛域

核心逻辑：收敛半径决定区间，端点单独判别。

1. 求收敛半径 R\boldsymbol{R}R

对幂级数 ∑n=0∞an(x−x0)n\boldsymbol{\sum_{n=0}^{\infty} a_n (x-x_0)^n}∑n=0∞​an​(x−x0​)n：

  • 特殊形式（如 ∑anx2n\sum a_n x^{2n}∑an​x2n）：视为 ∑(anx2)n\sum (a_n x^2)^n∑(an​x2)n，用根值法得 ρ∣x∣2<1\rho |x|^2 < 1ρ∣x∣2<1，故 R=1/ρR = 1/\sqrt{\rho}R=1/ρ​。

方法选择优先级

  • 幂级数求和：优先 凑麦克劳林公式（最快）→ 其次 逐项积分/求导（通用）→ 最后 微分方程（复杂递推型）。
  • 常数项级数求和：优先 定义法（裂项、等差等比）→ 其次 幂级数间接法（需构造级数并验证端点收敛性）。

通过“识别结构→匹配方法→验证收敛域”，逐步拆解问题，避免盲目运算！

五、函数展开成幂级数（间接法为主）

核心逻辑：通过变量代换、四则运算、逐项求导/积分，将目标函数转化为已知展开式。

1. 步骤

① 选展开中心（默认 x=0x=0x=0，即麦克劳林级数；若需 x=x0x=x_0x=x0​，作代换 t=x−x0t = x - x_0t=x−x0​）；
② 变形目标函数：

  • 分式型（如 1a−bx\frac{1}{a - bx}a−bx1​）：拆为 1a⋅11−bx/a\frac{1}{a} \cdot \frac{1}{1 - bx/a}a1​⋅1−bx/a1​，用 1/(1−x)1/(1-x)1/(1−x) 的展开；
  • 对数/反三角型（如 ln⁡(1+x2)\ln(1+x^2)ln(1+x2)）：求导转化为分式，展开后积分；
  • 三角函数型（如 sin⁡2x\sin 2xsin2x）：变量代换（x→2xx\to 2xx→2x）；
    ③ 确定收敛域：由已知展开式的收敛域和变形限制共同决定（如代换 t=x2t = x^2t=x2 则 ∣t∣<1⇒∣x∣<1|t| < 1 \Rightarrow |x| < 1∣t∣<1⇒∣x∣<1）。
2. 易错点
  • 直接法（求泰勒系数）计算复杂，仅用于证明或特殊函数，优先间接法；
  • 展开后标注收敛域，尤其端点（如 ln⁡(1+x)\ln(1+x)ln(1+x) 在 x=1x=1x=1 处收敛，需包含）。

六、傅里叶级数（数学一）

核心逻辑：周期延拓→求系数→应用狄利克雷定理求和函数。

1. 周期函数（周期 2l2l2l）
  • 求系数：
    {a0=1l∫−llf(x)dx,an=1l∫−llf(x)cos⁡nπxldx (n≥1),bn=1l∫−llf(x)sin⁡nπxldx (n≥1). \begin{cases} a_0 = \dfrac{1}{l} \int_{-l}^{l} f(x)dx, \\ a_n = \dfrac{1}{l} \int_{-l}^{l} f(x)\cos\dfrac{n\pi x}{l}dx \ (n\geq 1), \\ b_n = \dfrac{1}{l} \int_{-l}^{l} f(x)\sin\dfrac{n\pi x}{l}dx \ (n\geq 1). \end{cases} ⎩⎨⎧​a0​=l1​∫−ll​f(x)dx,an​=l1​∫−ll​f(x)coslnπx​dx (n≥1),bn​=l1​∫−ll​f(x)sinlnπx​dx (n≥1).​
  • 狄利克雷定理：和函数 S(x)=f(x−)+f(x+)2S(x) = \dfrac{f(x^-) + f(x^+)}{2}S(x)=2f(x−)+f(x+)​（x−x^-x− 左极限，x+x^+x+ 右极限）；连续点处 S(x)=f(x)S(x) = f(x)S(x)=f(x)，间断点处取左右平均。
2. 非周期函数（区间 [0,l][0,l][0,l] 或 [−l,l][-l,l][−l,l]）
  • 奇延拓（展开为正弦级数）：f(x)→f(x) \tof(x)→ 奇函数，故 an=0a_n = 0an​=0，bn=2l∫0lf(x)sin⁡nπxldxb_n = \dfrac{2}{l} \int_{0}^{l} f(x)\sin\dfrac{n\pi x}{l}dxbn​=l2​∫0l​f(x)sinlnπx​dx；
  • 偶延拓（展开为余弦级数）：f(x)→f(x) \tof(x)→ 偶函数，故 bn=0b_n = 0bn​=0，an=2l∫0lf(x)cos⁡nπxldxa_n = \dfrac{2}{l} \int_{0}^{l} f(x)\cos\dfrac{n\pi x}{l}dxan​=l2​∫0l​f(x)coslnπx​dx。
3. 步骤

① 确定周期和延拓方式（奇/偶/周期延拓）；
② 利用奇偶性简化积分（如奇函数在对称区间积分=0，偶函数加倍半区间积分）；
③ 写出傅里叶级数，用狄利克雷定理确定和函数（尤其端点 x=0,±lx=0, \pm lx=0,±l 和间断点）。

总结：核心思维

  • 敛散性：类型化处理，抓“正项→比较/比值，交错→莱布尼茨，任意→绝对收敛优先”；
  • 幂级数：收敛半径是桥梁，求和与展开互逆，依托已知展开式变形；
  • 傅里叶：积分求系数，狄利克雷定理判和函数，注意延拓和间断点。

通过“分类→方法→步骤→易错点”拆解，逐个突破题型！
